MultiPSK2RdsSpy - VS 2015 solution 
C#, Visual Studio 2015 Community, .NET 4.0. 
http://teknokoodiradio.vuodatus.net/lue/2016/04/multipsk2rdsspy-multipsk-sta-rds-data-rds-spy-ohjelmaan
MultiPSK RDS decoded blocks to RDS Spy *.spy file format. 
At first start MultiPSK RDS & decoded blocks and then start MultiPSK TCP/IP link. 
Set same TCP port in MultiPSK & this app. Press Start and select file to save data.
